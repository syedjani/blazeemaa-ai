# 🤖 Blazeemaa AI

Blazeemaa AI is a chatbot built using Google's Gemini API and Gradio.

## 🚀 Features
- AI-powered chat
- Simple and clean UI
- Deployable on Hugging Face / Render
- Uses Gemini model

## 🛠 Tech Stack
- Python
- Gradio
- Google Gemini API

## ⚙️ Setup

### 1. Clone the repo
git clone https://github.com/yourusername/blazeemaa-ai.git  
cd blazeemaa-ai  

### 2. Install dependencies
pip install -r requirements.txt  

### 3. Add API Key
Create `.env` file:

GOOGLE_API_KEY=your_api_key_here  

### 4. Run the app
python app.py  

## 🌐 Deployment
You can deploy this project on:
- Hugging Face Spaces
- Render
- Railway
